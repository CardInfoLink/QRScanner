# QRScanner
二维码扫码优化


## 解码优化
1. 减少解码格式
2. 解码算法优化
3. 减少解码数据
4. 解码库Zbar 与Zxing融合

## 优化相机设置
1. 选择最佳预览尺寸/图片尺寸
2. 设置适合的相机放大倍数
3. 调整聚焦时间
4. 设置自动对焦区域
5. 调整合理扫描区域
